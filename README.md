# AI-self-taught

## Purpose | 项目目的

This repository is created to record my self-learning journey in Artificial Intelligence.  
It is mainly used to keep the code I write during learning, testing, and practice, so that I can track my progress and build a clearer understanding step by step.

这个仓库用于记录我在人工智能方向上的自学过程。  
它主要用来保存我在学习、尝试和练习过程中编写的代码，方便我整理思路、回顾过程，并一步步加深理解。

这个仓库后续可能会逐步包含：
